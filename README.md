
# Sistema de Vendas - TCC

## Descrição
Este projeto foi desenvolvido como parte do Projeto Integrado 1 (UNIVESP).
O sistema tem como objetivo gerenciar vendas de um bar utilizando o framework Django.

## Funcionalidades
- Cadastro de produtos
- Registro de vendas
- Controle básico de sistema

## Tecnologias utilizadas
- Python
- Django
- SQLite

## Como executar o projeto

1. Instalar dependências:
pip install -r requirements.txt

2. Rodar o servidor:
python manage.py runserver

3. Acessar no navegador:
http://127.0.0.1:8000/
